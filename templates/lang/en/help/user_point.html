Description

'Point' is a value to describe the user.

How can I increasing my point?

  • +2 for posting a topic, -2 for deleting one
  • +1 for posting a reply, -1 for deleting one
  • +1 for login
  • +4 for eliting a topic, -4 for uneliting one
